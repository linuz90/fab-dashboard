# Geist Pixel

`GeistPixel-Latin.woff2` is the Latin webfont served by the official Google Fonts specimen for [Geist Pixel](https://fonts.google.com/specimen/Geist+Pixel). It is self-hosted so themes work offline and do not make third-party font requests.
